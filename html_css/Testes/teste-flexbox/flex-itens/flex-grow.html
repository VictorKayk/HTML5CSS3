flex-grow: ;

Usar para definir a proporção em que cada Flex Item irá crescer se houver espaço restante, os valores são definidos com números (o justify-content: ; não irá funcionar com o flex-grow: ; diferente de 0).

flex-grow: 0; (Padrão)

flex-grow: 0; (Padrão), os Flex Itens não irão crescer, eles ficarão com o espaço necessario para o seu conteudo.

flex-grow: ; com o mesmo valor em todos os itens.

Quando todos os Flex Itens recebem o mesmo valor, diferente de 0, eles são ajustados para ocupar totalmente o conteiner, com a mesma proporção de tamanho para cada (se um item tiver um conteudo maior que os outros, ele ocupará um espaço maior do conteiner, mas receberá um crescimento do flex-grow: ; do mesmo tamanho que os outros).

flex-grow: ; com valores diferentes.

Quando os Flex Itens recebem valores diferentes, eles são ajustados para crescer proporcionalmente aos valores que receberam, vizando ocupar o conteiner totalmente. Usando o exemplo a seguir: Os Flex Itens 1, 3 e 5 receberam o flex-grow: 1;, e os Flex Itens 2 e 4 receberam o flex-grow: 2;, os itens 2 e 4 serão ajustados para ocupar 2 vezes mais o conteiner do que os itens 1, 3 e 5. Se eles tivessem recebido o flex-grow: 3; em vez do 2, seriam ajustados para ocupar 3 vezes mais que os outros, e assim por diante, seguindo a proporção.



flex-grow: 1; nos itens 1, 3 e 5 / flex-grow: 2; nos itens 2 e 4.
