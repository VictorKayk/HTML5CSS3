Flex Itens.

Flex Itens são os filhos dos Flex Conteiners. Os Flex Itens também podem ser conteiners, bastas adicionar uma propriedade display: flex; a ele (Suas tags filhas irão se tornar Flex Itens também).

flex-grow: ;

Usar para definir a proporção em que cada Flex Item irá crescer, os valores são definidos com números (o justify-content: ; não irá funcionar com o flex-grow: ; diferente de 0).

flex-grow: 0; (Padrão), os Flex Itens não irão crescer, eles ficarão com o espaço necessario para o seu conteudo.

flex-grow: ; com o mesmo valor em todos os itens. Quando todos os Flex Itens recebem o mesmo valor, diferente de 0, eles são ajustados para ocupar totalmente o conteiner, com a mesma proporção de tamanho para cada (se um item tiver um conteudo maior que os outros, ele ocupará um espaço maior do conteiner, mas receberá um crescimento do flex-grow: ; do mesmo tamanho que os outros).

flex-grow: ; com valores diferentes. Quando os Flex Itens recebem valores diferentes, eles são ajustados para crescer proporcionalmente aos valores que receberam, vizando ocupar o conteiner totalmente.
