Flex Itens.

Flex Itens são os filhos dos Flex Conteiners. Os Flex Itens também podem ser conteiners, bastas adicionar uma propriedade display: flex; a ele (Suas tags filhas irão se tornar Flex Itens também).

flex-grow: ;

Usar para definir a proporção em que cada Flex Item irá crescer, os valores são definidos com números (o justify-content: ; não irá funcionar com o flex-grow: ; diferente de 0).

flex-grow: 0; (Padrão), os Flex Itens não irão crescer, eles ficarão com o espaço necessario para o seu conteudo.

flex-grow: ; com o mesmo valor em todos os itens. Quando todos os Flex Itens recebem o mesmo valor, diferente de 0, eles são ajustados para ocupar totalmente o conteiner, com a mesma proporção de tamanho para cada (se um item tiver um conteudo maior que os outros, ele ocupará um espaço maior do conteiner, mas receberá um crescimento do flex-grow: ; do mesmo tamanho que os outros).

flex-grow: ; com valores diferentes. Quando os Flex Itens recebem valores diferentes, eles são ajustados para crescer proporcionalmente aos valores que receberam, vizando ocupar o conteiner totalmente.

flex-basis: ;

Usar para definir o espaço inicial dos Flex Itens, antes de distribuir o espaço restante (se usar o flex-grow: ; em conjunto) (de maneira grosseira, o flex-basis: ; funciona como um width e height para os Flex Itens). Pode-se definir um tamanho fixo para os Flex Itens, que será respeitado se o conteudo não for maior, ou se o item não estourar o conteiner (sem wrap). Na maioria das vezes é usado em conjunto com outras propriedades do Flex Item. Algumas maneiras de configurar o flex-basis: ;, são:

flex-basis: auto; (Padrão), definirar o espaço, como literalmente o tamanho do conteudo dos Flex Itens.

O flex-basis: ; (com tamanho fixo), irá literalmente criar um tamanho fixo para todos os itens definidos com ele, mas só terá efeito se: Ouver espaço para que os itens definidos pelo flex-basis: ; fiquem dentro do conteiner (ele irá ignorar o tamanho definido se os itens estourarem o conteiner, e não haver um wrap para que sejam acomodos em outra linha) . Ou quando o tamanho definido pelo flex-basis: ; não for menor que o conteudo.

A união do flex-basis: ; e do flex-grow: ; fara com o ques os Flex Itens tentem manter um valor padrão (dependendo do tamanho do conteudo).

Se os flex-basis: ; tiverem o mesmo valor, eles tentarão ocupar todo o conteiner com a mesma proporção (respeitando o tamanho do conteudo), mas se os valores forem diferentes, eles tentarão ocupar todo o conteiner com a proporção diferente (como o flex-grow: ;).
Se os flex-basis: ; tiverem um valor definido e tiver a adicionado um flex-grow: ;, o Flex Item terá um tamanho fixo, só que tentará ocupar todo o conteiner se sobrar espaço (ele respeitará as limitações tamanho do conteudo).
