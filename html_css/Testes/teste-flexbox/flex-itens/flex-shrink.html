flex-shrink: ;

Usar para definir a proporção em que cada Flex Item, irá diminuir se não houver mais espaço. Os valores são definidos com números (É basicamente, o contrario do flex-grow: ;). Na maioria das vezes é usado em conjunto com outras propriedades do Flex Item.

flex-shrink: 1; (Padrão)

flex-shrink: 1; (padrão), indica que todos os Flex Itens irão diminuir na mesma proporção. E que os itens, com tamanho definido, poderão diminuir para caber em um Flex conteiner.
