flex-basis: ;

Usar para definir o espaço inicial dos Flex Itens, antes de distribuir o espaço restante (se usar o flex-grow: ; em conjunto) (de maneira grosseira, o flex-basis: ; funciona como um width e height para os Flex Itens). Pode-se definir um tamanho fixo para os Flex Itens, que será respeitado se o conteudo não for maior, ou se o item não estourar o conteiner (sem wrap). Na maioria das vezes é usado em conjunto com outras propriedades do Flex Item.

(Todos os Flex Itens a seguir foram estabelicidos com o mesmo valor, só que a graça os Flex Itens é utilizar valores diferentes em cada)

auto (Padrão)

flex-basis: auto; (Padrão), definirar o espaço, como literalmente o tamanho do conteudo dos Flex Itens.

Definindo um tamanho de flex-basis: ; (fixo).

O flex-basis: ; (com tamanho fixo), irá literalmente criar um tamanho fixo para todos os itens definidos com ele, mas só terá efeito se: Houver espaço para que os itens definidos pelo flex-basis: ; fiquem dentro do conteiner (ele irá ignorar o tamanho definido se os itens estourarem o conteiner, e não haver um wrap para que sejam acomodos em outra linha) . Ou quando o tamanho definido pelo flex-basis: ; não for menor que o conteudo.

União do flex-basis: ; e do flex-grow: ;

A união do flex-basis: ; e do flex-grow: ; fara com o ques os Flex Itens tentem manter um valor padrão (dependendo do tamanho do conteudo).

Se os flex-basis: ; tiverem o mesmo valor, eles tentarão ocupar todo o conteiner com a mesma proporção (respeitando o tamanho do conteudo), mas se os valores forem diferentes, eles tentarão ocupar todo o conteiner com a proporção diferente (como o flex-grow: ;).


flex-basis: 0; // flex-grow: 1;
