align-self: ;

Usar para definir o alinhamento especifico de cada Flex Item individualmente dentro do conteiner (basicamente, tem a mesma função da propriedade align-items: ;, só que para itens especificos e não para o conteiner todo). Quando definido, sobrescreve o valor da propriedade aling-items: ;, que define o alinhamento do conteiner em geral (funciona com qualquer flex-direction: ;)(de forma grosseira, alinha o flex-direction: row; verticalmente e o flex-direction: column; horizontalmente).

auto (Padrão)

Valor padrão, quando definido esse valor para propriedade aling-self: ;, ou quando não definido nenhum valor, o Flex Item irá seguir o alinhamento do conteiner.
