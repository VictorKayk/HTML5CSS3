flex: ;

flex: ; é uma propriedade que une as propriedades flex-grow: ;, flex-shrink e flex-basis (nessa ordem), em uma só (um atalho). Ela permite todas as combinações de valores das 3 propriedades. Por quesito de compatibilidade nos navegadores, é recomendado que use esse atalho em vez das 3 propriedades sozinhas. Algumas dessa combinações possiveis são:

flex: 0 1 auto; (Padrão)

Se a propriedade flex for adicionada com esse valor ou não , nada irá mudar.
