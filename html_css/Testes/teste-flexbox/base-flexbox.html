Base do FlexBox.

display: flex;

Usar para transformar um elemento em um Flex Conteiner, e todos seus filhos diretos (Todas as tags que estão diretamente dentro da tag que virou um Flex Conteiner) em Flex Itens.

Flex Conteiner.

Flex Conteiner é o elemento que envolve todos os Flex Itens. Ele é criado, e só se torna "moldável" por causa da propriedade display: flex; adicionada a ele.

Flex Itens.

Flex Itens são os filhos dos Flex Conteiners. Os Flex Itens também podem ser conteiners, bastas adicionar uma propriedade display: flex; a ele (Suas tags filhas irão se tornar Flex Itens também).
