Flex Conteiner.

Usar para transformar um elemento em um Flex Conteiner, e todos seus filhos diretos (Todas as tags que estão diretamente dentro da tag que virou um Flex Conteiner) em Flex Itens.

flex-direction: ;

Usar para definir a direção dos Flex Itens, dentro do Flex Conteiner. É possivel usar alguns valores para mudar essa direção, como:
row (padrão), deixa os itens alinhados horizontalmente em linha.
row-reverce, deixa os itens alinhados horizontalmente em linha, porem em uma linha reversa (Começa do ultimo e termina no primeiro).
column, deixa os itens alinhados verticalmente em coluna.
column-reverce, deixa os itens alinhados verticalmente em coluna, porem em uma coluna reversa (Começa do ultimo e termina no primeiro).

flex-wrap: ;

Usar para definir se dentro do Flex Conteiner haverá ou não a quebra de linha quando um Flex Item ultrapassar o tamanho do conteiner. É possivel usar alguns valores para configurar essa quebra de linha, como:
nowrap (Padrão), usar para definir que não haverá quebra de linha, quando um Flex Item ultrapassar o tamanho do conteiner.
wrap, usar para definir que o Flex Item irá quebrar a linha, quando ultrapassar o tamanho do conteiner (da maneira normal).
wrap-reverce, usar para definir que o Flex Item irá quebrar a linha, quando ultrapassar o tamanho do conteiner, porem irá quebrar de forma reversa, o item sobressalente irá para a linha cima, em vez da linha de baixo como padrão.
