Flex Conteiner

Usar para transformar um elemento em um Flex Conteiner, e todos seus filhos diretos (Todas as tags que estão diretamente dentro da tag que virou um Flex Conteiner) em Flex Itens.

flex-direction: ;

Usar para definir a direção dos Flex Itens, dentro do Flex Conteiner. É possivel usar alguns valores para mudar essa direção, como:

row (padrão), deixa os itens alinhados horizontalmente em linha.

row-reverce, deixa os itens alinhados horizontalmente em linha, porem em uma linha reversa (Começa do ultimo e termina no primeiro).

column, deixa os itens alinhados verticalmente em coluna.

column-reverce, deixa os itens alinhados verticalmente em coluna, porem em uma coluna reversa (Começa do ultimo e termina no primeiro).

flex-wrap: ;

Usar para definir se dentro do Flex Conteiner haverá ou não quebra de linha quando um Flex Item ultrapassar o tamanho do conteiner. É possivel usar alguns valores para configurar essa quebra de linha, como:

nowrap (Padrão), usar para definir que não haverá quebra de linha, quando um Flex Item ultrapassar o tamanho do conteiner.

wrap, usar para definir que o Flex Item irá quebrar a linha, quando ultrapassar o tamanho do conteiner (da maneira normal).

wrap-reverce, usar para definir que o Flex Item irá quebrar a linha, quando ultrapassar o tamanho do conteiner, porem irá quebrar de forma reversa/invertida, o item sobressalente irá para a linha cima, em vez da linha de baixo como padrão.

flex-flow: ;

flex-flow: ; é um propriedade que une as propriedades flex-direction: ; e flex-wrap em uma só (um atalho). Ela permite todas as combinações de valores das duas propriedade, e não necessita de uma ordem especifica.

justify-content: ;

Usar para alinhar os Flex Itens dentro do Flex Conteiner. Os itens não podem ocupar todo o espaço do conteiner (funciona com qualquer flex-direction: ;)(de forma grosseira, alinha o flex-direction: row; horizontalmente e o flex-direction: column; verticalmente). É possivel usar alguns valores para configurar esse espaçamento, como:

flex-start (Padrão), alinha os itens no começo do conteiner (a esquerda para o flex-direction: row; e em cima para o flex-direction: column;).

flex-end, alinha os itens no fim do conteiner (a direita para o flex-direction: row; e em baixo para o flex-direction: column;).

center, como o proprio nome já diz, alinha os itens ao centro do conteiner (no centro horizontal para flex-direction: row; e no centro vertical para o flex-direction: column;).

space-between, alinha os itens dentro do conteiner, com um espaço igual entre eles. O primeiro item fica alinhado no inicio do conteiner e o ultimo item fica alinhado no fim do conteiner.

space-around, alinha os itens dentro do conteiner, com um espaço entorno deles. O espaço entre os itens é o dobro do espaço inicial e do espaço final.

space-evenly, alinha os itens dentro do conteiner, com um espaço entorno deles. Todos os espaços entorno dos itens, tem o mesmo tamanho.

align-items: ;

Usar para alinhar os Flex Itens dentro do Flex Conteiner(funciona com qualquer flex-direction: ;)(de forma grosseira, alinha o flex-direction: row; verticalmente e o flex-direction: column; horizontalmente). É possivel usar alguns valores para configurar esse alinhamento, como:

stretch (Padrão), estica os Flex Itens igualmente, e faz com que o flex-direction: row; preencha o conteiner verticalmente e o flex-direction: column; preencha o conteiner horizontalmente.

flex-start, alinha os itens no começo do conteiner (em cima para o flex-direction: row; e a esquerda para o flex-direction: column;).

flex-end, alinha os itens no fim do conteiner (em baixo para o flex-direction: row; e a direita para o flex-direction: column;).

center, como o proprio nome já diz, alinha os itens ao centro do conteiner (no centro vertical para flex-direction: row; e no centro horizontal para o flex-direction: column;).

baseline, alinha os itens no conteiner, de acordo com a baseline da fonte das letras (verticalmente para flex-direction: row; e horizontalmente para o flex-direction: column;).

align-content: ;

Usar para modificar/definir o comportamento das linhas do Flex Conteiner, definidas pelo flex-wrap: ;. Semelhante ao align-items: ;, mas em vez de alinhar todos os itens flexiveis individualmente, ele alinha as linhas flexiveis do conteiner individualmente (só funciona se o flex-wrap: ; for diferente de nowrap). É possivel usar alguns valores para configurar esse alinhamento das linhas, como:

stretch (Padrão), estica os Flex Itens igualmente, e faz com que o flex-direction: row; preencha o conteiner verticalmente e o flex-direction: column; preencha o conteiner horizontalmente.

flex-start, alinha os itens no começo do conteiner (em cima para o flex-direction: row; e a esquerda para o flex-direction: column;).

flex-end, alinha os itens no fim do conteiner (em baixo para o flex-direction: row; e a direita para o flex-direction: column;).

center, como o proprio nome já diz, alinha os itens ao centro do conteiner (no centro vertical para flex-direction: row; e no centro horizontal para o flex-direction: column;).

space-between, alinha os itens dentro do conteiner, com um espaço igual entre eles. O primeiro item fica alinhado no inicio do conteiner e o ultimo item fica alinhado no fim do conteiner.

space-around, alinha os itens dentro do conteiner, com um espaço entorno deles. O espaço entre os itens é o dobro do espaço inicial e do espaço final.

space-evenly, alinha os itens dentro do conteiner, com um espaço entorno deles. Todos os espaços entorno dos itens, tem o mesmo tamanho.
