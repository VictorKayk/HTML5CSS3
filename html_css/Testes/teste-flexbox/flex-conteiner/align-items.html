align-items: ;

Usar para alinhar os Flex Itens dentro do Flex Conteiner(funciona com qualquer flex-direction: ;)(de forma grosseira, alinha o flex-direction: row; verticalmente e o flex-direction: column; horizontalmente).

stretch (Padrão)

stretch (Padrão), estica os Flex Itens igualmente, e faz com que o flex-direction: row; preencha o conteiner verticalmente e o flex-direction: column; preencha o conteiner horizontalmente.
