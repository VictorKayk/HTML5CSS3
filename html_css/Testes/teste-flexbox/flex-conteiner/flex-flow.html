flex-flow: ;

flex-flow: ; é um propriedade que une as propriedades flex-direction: ; e flex-wrap em uma só (um atalho). Ela permite todas as combinações de valores das duas propriedade, e não necessita de uma ordem especifica. Algumas dessa combinações possiveis são:

row nowrap

1
2
3
4
5

row wrap

1
2
3
4
5

column nowrap

1
2
3
4
5
